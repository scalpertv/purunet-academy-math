# 나칸 중1 수학 워크북 — RealWorld 프로토타입

간단한 실감형 체험 데모입니다. A-Frame을 이용해 브라우저에서 3D 오브젝트를 조작합니다.

빠른 시작

1. 폴더를 로컬에서 열고 `index.html`을 브라우저에서 엽니다.
2. 오브젝트를 클릭하거나 우측 컨트롤을 사용해 크기를 조절하세요.

배포

정적 사이트이므로 GitHub Pages / Cloudflare Pages에 그대로 올리면 됩니다. 원본 워크북 내용(문제/이미지)을 `content/` 폴더로 옮겨 시나리오와 연결하세요.

다음 단계 제안

- 문제에 맞춘 체험 시나리오(분수, 도형 넓이 비교, 좌표 이동)를 추가
- Three.js로 복잡한 시뮬레이션(광원, 물리 기반 애니메이션) 구현
