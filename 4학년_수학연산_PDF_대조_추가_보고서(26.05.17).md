## 이번 보강 내용

1. 4학년 연산 전용 토픽 `g4-op-*`에 시각자료 자동 보강 로직을 추가했다.
2. 이미 그래프나 도형이 있는 문제는 기존 시각자료를 유지하고, 텍스트만 있던 문제에만 보강 자료를 붙이도록 했다.
3. 새 벡터 시각자료 타입을 추가했다.
   - `range-line`: 이상·이하·초과·미만을 수직선으로 표시
   - `parallel-lines`: 평행, 수직, 평행선 사이 거리 표시
   - `quadrilateral-diagram`: 사다리꼴, 평행사변형, 마름모, 직사각형, 정사각형 표시
   - `polygon-diagram`: 다각형, 정다각형, 대각선 표시
4. 검증 스크립트도 새 시각자료 타입을 검사할 수 있도록 갱신했다.

## 최종 판정

- 4학년 연산 3월~2월 전체 흐름은 PDF 월별 내용과 대응된다.
- 수학적 도형, 도표, 표, 막대그래프, 꺾은선그래프는 앱 문제 생성 로직에 반영되어 있다.
- 기존에 숫자와 문장만 나오던 일부 4학년 연산 문제도 자리값 표, 분수 막대, 범위 수직선, 평행선·수선, 사각형·다각형 벡터 그림으로 보강했다.
