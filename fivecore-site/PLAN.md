# 5차원 자기주도학습 사이트 구축 계획

> 기반 문서: 원동연 박사 5차원 전면교육 커리큘럼(26.06.10).md  
> 배포 URL: https://fivecore-self-directed-learning.pages.dev  
> 작성일: 2026-06-10

---

## 기술 스택

- **프레임워크:** React 19 + TypeScript + Vite (기존 app과 동일)
- **스타일:** Tailwind CSS (새 프로젝트이므로 도입)
- **상태 저장:** localStorage (MVP) → Cloudflare D1 (확장)
- **배포:** Cloudflare Pages (새 프로젝트: `fivecore-self-directed-learning`)
- **차트:** SVG 직접 구현 (레이더 차트)

---

## 전체 페이지 구조

```
/                       → 메인 (학년군 선택 랜딩)
/guide/:grade           → 학년군별 5단계 사이클 가이드
/planner                → 디지털 학습 플래너
/reading                → 속해독서 타이머 + 기록
/analysis               → 글분석 도구
/mindmap                → 마인드맵 도구
/dashboard              → 5차원 성장 대시보드
/portfolio              → 포트폴리오 보관함
/curriculum/:subject    → 과목별 커리큘럼 가이드
```

## Phase 1 — 프로젝트 초기 설정 및 랜딩 페이지

**범위:** 새 프로젝트 scaffold + 메인 화면

### 작업 목록
1. `fivecore-site/` 디렉터리 생성
2. `npm create vite@latest . -- --template react-ts` 실행
3. Tailwind CSS 설치 및 설정
4. `wrangler.toml` 작성 (project-name: `fivecore-self-directed-learning`)
5. 메인 랜딩 페이지 (`/`) 구현
   - 사이트 제목: "5차원 자기주도학습"
   - 5차원 소개 카드 5개 (지적역량/정서강건성/신체강건성/자기관리/인간관계)
   - 학년군 선택 버튼 7개 (6세반/7세반/초등저/초등고/중등/고등)
   - 선택 → `/guide/:grade` 이동
6. 공통 레이아웃 (헤더 + 하단 네비게이션 탭바)
7. 색상 시스템 정의 (5차원별 컬러 팔레트)

**검증 기준:** `npm run dev` 실행 시 메인 페이지 렌더링 정상, 학년군 클릭 → 가이드 페이지 이동

## Phase 2 — 학년군별 5단계 사이클 가이드 페이지

**범위:** `/guide/:grade` — 오늘의 5단계 학습 흐름 안내

### 작업 목록
1. 학년군 데이터 파일 `src/data/grades.ts` 작성
   - 7개 학년군별 하루 학습 흐름 (총 소요 시간, 단계별 분 수)
   - 단계별 설명 텍스트 (커리큘럼 문서 3장 내용)
2. `GuidePage` 컴포넌트 구현
   - 학습 흐름 타임라인 UI (5단계 세로 스텝)
   - 각 단계 클릭 → 확장 설명 패널
   - "오늘 시작하기" 버튼 → 플래너 페이지 이동
3. 6세/7세반 전용 UI (그림 카드, 이모지 중심)
4. 고등반 전용 UI (수능 연계 배지 표시)

**검증 기준:** 7개 학년군 각각 다른 소요시간과 단계 설명 표시

## Phase 3 — 디지털 학습 플래너

**범위:** `/planner` — 오늘의 목표 설정·실행·성찰

### 작업 목록
1. `src/data/planner.ts` — 학년군별 플래너 양식 데이터
2. 플래너 폼 컴포넌트
   - 6~7세: 그림 카드 선택 UI
   - 초등 저: 목표 3개 텍스트 입력 + 예상 시간
   - 초등 고: SMART 목표 양식 (과목/단원/구체적목표/측정기준/예상시간)
   - 중·고등: SMART + D-Day 역산 + 에너지 레벨 조절
3. 5단계 사이클 체크박스 (플래너/묵상/독서/글분석/마인드맵)
4. 신체 기록 섹션 (수면/운동/물/아침식사)
5. 성찰 섹션 (잘한것/아쉬운것/내일개선)
6. localStorage 저장 (날짜별 키)
7. 달성률 자동 계산 및 표시 (%)
8. 오늘 플래너 / 이전 플래너 탭 전환

**검증 기준:** 저장 후 새로고침해도 오늘 데이터 유지, 달성률 정확히 계산

## Phase 4 — 속해독서 타이머

**범위:** `/reading` — 읽기 타이머 + 독해 속도 계산 + 독서 기록

### 작업 목록
1. 타이머 컴포넌트
   - 시작 / 일시정지 / 정지 버튼
   - 경과 시간 MM:SS 표시
   - 시각적 진행 원형 애니메이션
2. 독해 속도 계산
   - 종료 후 읽은 글자 수 입력 → 분당 글자 수 자동 계산
   - 학년군 목표 속도 대비 달성률 표시
   - 예: "초3~4 목표: 분당 300자 | 오늘: 285자 (95%)"
3. 속해독서 기록지 양식 (읽기 전 예측 / 읽은 후 확인 / 새로 알게 된 것 / 궁금한 것)
4. 독서 기록 로그
   - 날짜 / 제목 / 읽은 시간 / 분당 속도 / 3줄 요약
   - localStorage 누적 저장
5. 이번 달 독서 통계 (권 수, 평균 속도, 총 독서 시간)

**검증 기준:** 타이머 시작→정지 후 글자 수 입력 시 분당 속도 정확히 계산

---

## Phase 5 — 글분석 도구

**범위:** `/analysis` — 학년군별 글분석 양식 제공 및 저장

### 작업 목록
1. `src/data/analysis.ts` — 학년군별 글분석 틀 데이터
   - 6세: 이야기 순서 카드 (처음/중간/끝)
   - 7세: 핵심 단어 + 원인-결과
   - 초등 저: 핵심어/글 종류/주제문
   - 초등 고: 논리 구조/사실vs의견/비판적 읽기
   - 중등: 논설문 분석 6단계 틀
   - 고등: 수능 비문학 분석 틀 6단계
2. 글분석 폼 컴포넌트 (학년군에 따라 필드 자동 변경)
3. 영어 글분석 탭 (저학년 4-question / 고학년 6-field)
4. 제목 + 날짜 + 글 종류 메타데이터 입력
5. localStorage 저장 (날짜별, 최근 10개 목록)
6. 저장된 분석 목록 조회 + 상세 보기

**검증 기준:** 학년군 전환 시 폼 필드가 즉시 교체됨

---

## Phase 6 — 마인드맵 도구

**범위:** `/mindmap` — 트리 구조 마인드맵 입력·저장·보기

### 작업 목록
1. 마인드맵 데이터 구조 설계 (JSON 트리: id/text/color/children)
2. 중심 노드 + 가지 노드 입력 UI
   - 중심 주제 텍스트 입력
   - "+ 가지 추가" 버튼 (최대 7개)
   - 각 가지에 "+ 세부 항목 추가" (2~3개)
   - 가지별 색상 선택 (6색 팔레트)
3. SVG 기반 마인드맵 렌더러
   - 중심 원 → 가지 선 → 말단 노드 시각화
   - 학년군별 복잡도 (6세: 가지 3개 / 고등: 계층 3단)
4. 마인드맵 템플릿 (학년군별 예제 제공)
   - 예: 초등 3학년 "곱셈이란?" 템플릿
5. PNG 저장 (canvas toDataURL)
6. localStorage 저장 (최근 10개)
7. 저장 목록 갤러리 보기

**검증 기준:** 가지 추가/삭제/색상 변경 즉시 SVG에 반영

## Phase 7 — 5차원 성장 대시보드

**범위:** `/dashboard` — 레이더 차트 + 주간 달성률 + 연속 학습 스트릭

### 작업 목록
1. 오늘의 학습 사이클 완성도 카드
   - 5단계 아이콘 (플래너/묵상/독서/글분석/마인드맵) 체크 상태
   - 오늘 플래너 달성률 연동
2. 5차원 레이더 차트 (SVG 직접 구현)
   - 5개 축: 지적역량/정서강건성/신체강건성/자기관리/인간관계
   - 학생 자기 평가 슬라이더 (1~5점) → 차트 실시간 업데이트
   - 이번 달 vs 지난 달 비교 선 표시
3. 이번 주 달성률 진행 바
4. 연속 학습 일수 스트릭 카운터 (불꽃 아이콘)
5. 이번 달 독서 통계 (독서 타이머 데이터 연동)
6. 최근 마인드맵 3개 미리보기 썸네일
7. 주간 신체 기록 요약 (수면/운동/물)

**검증 기준:** 플래너 저장 → 대시보드 달성률 자동 반영

## Phase 8 — 포트폴리오 보관함

**범위:** `/portfolio` — 학기별 누적 저장 및 성장 기록

### 작업 목록
1. 포트폴리오 구조 (학기별 컨테이너)
   - 현재 학기: 2026-1학기
2. 포트폴리오 구성 요소 뷰어
   - ① 표지 (이름/학년/기간/다짐)
   - ② 5차원 성장 그래프 (학기 초 vs 학기 말)
   - ③ 베스트 플래너 3개 선택
   - ④ 베스트 마인드맵 3개 선택
   - ⑤ 베스트 글분석 2개 선택
   - ⑥ 독서 기록 목록
   - ⑦ 3분 묵상 특별 기록
   - ⑧ 자기성찰 에세이 입력란
3. 포트폴리오 저장 (localStorage → JSON export)
4. JSON import 기능 (다른 기기 복원)
5. 인쇄 최적화 CSS (포트폴리오 출력용)

**검증 기준:** 플래너/마인드맵/글분석에서 저장한 항목이 포트폴리오 선택 목록에 나타남

## Phase 9 — 과목별 커리큘럼 가이드

**범위:** `/curriculum/:subject` — 5과목 × 7학년군 커리큘럼 열람

### 작업 목록
1. `src/data/curriculum.ts` — 커리큘럼 문서 4장 전체 데이터화
   - 과목: 국어/수학/영어/사회/과학
   - 학년군 × 단원 목표 × 마인드맵 예시 × 속해독서 자료
2. 커리큘럼 탐색 UI
   - 과목 탭 + 학년군 드롭다운
   - 단원 목록 → 클릭 시 상세 내용 펼치기
3. 마인드맵 예시 연결 (클릭 → 마인드맵 도구에서 템플릿으로 열기)
4. 글분석 양식 연결 (클릭 → 글분석 도구에서 해당 틀로 열기)
5. 월별 성취 기준 체크포인트 표 (초4 예시 기준)
6. 연간 로드맵 요약표 (9장 내용)

**검증 기준:** 과목+학년군 조합 35가지 모두 고유 내용 표시

## Phase 10 — 3분 묵상 가이드 + 신체 기록 루틴

**범위:** 묵상 가이드 인터랙티브 + 신체 강건성 기록 전용 섹션

### 작업 목록
1. 3분 묵상 인터랙티브 가이드
   - 단계별 안내 (감사/회상/수용/다짐/조절) + 남은 시간 표시
   - 연령별 모드 (6세: 감정 색깔 카드 / 고등: 진로 연결)
   - 묵상 기록 저장 (오늘 감사한 것 / 다짐 텍스트)
2. 신체 강건성 전용 기록
   - 수면/운동/물/아침식사 일일 로그
   - 학년군별 목표 대비 달성 표시
   - 주간 신체 기록 히트맵
3. 부모/교사 코멘트 섹션 (localStorage 저장, PIN 잠금)

**검증 기준:** 묵상 타이머 3분 종료 후 기록 저장 정상 동작

## Phase 11 — Cloudflare Pages 배포 설정

**범위:** 실제 URL 배포 및 CI/CD 설정

### 작업 목록
1. `wrangler.toml` 작성 (project-name: `fivecore-self-directed-learning`)
2. `.github/workflows/deploy-fivecore.yml` 작성
3. `npm run build` → Cloudflare Pages 배포
4. 모바일 반응형 최종 점검 (iOS Safari, Android Chrome)
5. PWA manifest 추가 (홈 화면 추가 지원)
6. 오프라인 캐시 (Service Worker — 핵심 기능 오프라인 동작)

**검증 기준:** https://fivecore-self-directed-learning.pages.dev 접속 정상
