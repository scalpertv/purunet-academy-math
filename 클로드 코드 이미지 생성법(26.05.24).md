Anthropic의 Claude Code는 기본적으로 “코드 생성·자동화 에이전트”라서 직접 이미지 생성 모델이 내장되어 있지 않습니다.
하지만 실제 운영에서는 다음 4가지 방식으로 “단어 이미지 자동 생성 시스템”을 많이 구축합니다.

---

# 1. 가장 쉬운 방법 — Claude Code + Stable Diffusion API 연결

이 방식이 가장 현실적입니다.

구조:

```text
단어 리스트
   ↓
Claude Code
(프롬프트 자동 생성)
   ↓
Stable Diffusion API
   ↓
PNG 이미지 자동 저장
```

예시:

```python
word = "Apple"

prompt = f"""
Cute children's educational illustration of {word},
simple background,
storybook style,
high quality,
for elementary English ebook
"""
```

Claude Code가:

* 영어 단어 읽기
* 프롬프트 자동 생성
* API 호출 코드 작성
* 폴더 저장
* 파일명 정리
  까지 자동화합니다.

# 3. Leonardo AI 연결 방식

Leonardo AI API 사용 가능.

Claude Code가:

* CSV 읽고
* 프롬프트 생성하고
* Leonardo API 호출
* 다운로드 자동화

가능합니다.

* 품질 높음

```text
storybook illustration,

```

---

# Python 자동 생성 예시

```python
import requests

prompt = "Cute cartoon apple"

payload = {
    "prompt": prompt,
    "steps": 20
}

response = requests.post(
    "http://127.0.0.1:7860/sdapi/v1/txt2img",
    json=payload
)
```
