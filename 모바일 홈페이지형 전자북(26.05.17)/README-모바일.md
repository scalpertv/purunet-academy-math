# 강태훈샘 수학 익힘책 모바일 홈페이지형

이 폴더는 모바일 브라우저에서 바로 열기 좋은 홈페이지형 패키지입니다.

- `index.html`: 모바일 첫 화면
- `study.html`: 실제 수학 익힘책 앱
- `manifest.webmanifest`, `sw.js`: 웹 서버에 올렸을 때 홈 화면 추가와 오프라인 캐시 지원
- `열기-모바일홈.cmd`: Windows에서 모바일 홈 화면을 Edge로 바로 열기

휴대폰에서 사용하려면 이 폴더 전체를 웹 호스팅 또는 학교 서버에 올린 뒤 `index.html` 주소로 접속하면 됩니다.
