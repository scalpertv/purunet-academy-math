# 푸르넷 유치원 학습 놀이터 기획서

작성일: 2026-05-20

## 1. 목표

6세반과 7세반 아이들이 초등학교 입학 전에 필요한 국어, 수학, 영어 기초를 놀이, 카드북, 동화, 게임, 노래, 말하기 활동으로 자연스럽게 익히는 유치원생용 학습 놀이터를 만든다.

핵심 목표는 공부처럼 느껴지는 문제풀이보다, 아이가 직접 누르고 듣고 말하고 맞히고 꾸미는 경험을 통해 매일 10분에서 20분씩 반복 학습하도록 만드는 것이다.

## 2. 기본 방향

|---|---|---|---|

| 수학 | 수 세기, 모양, 비교, 분류, 규칙 찾기 중심 | 1부터 100까지 수, 덧셈·뺄셈 기초, 시계·돈·길이 중심 | 수 개념, 연산 준비, 생활 수학 |
| 영어 | 알파벳 소리, 그림 단어, 챈트, 몸동작 중심 | 파닉스, 생활 문장, 영어 동화 듣기, 따라 말하기 중심 | 영어 소리 친숙도, 기본 표현, 자신감 |
| 놀이 | 누르기, 끌어놓기, 스티커, 색칠, 짝찾기 | 미션, 보드게임, 카드 뒤집기, 말하기 녹음 | 집중 시간 증가, 규칙 따르기 |
| 평가 | 정답보다 참여 횟수와 반응 기록 | 정확도, 반복 횟수, 읽기·말하기 기록 | 초등 적응 리포트 |

- [ ] 7세반 영어 파닉스 카드북 3종을 만든다.
- [ ] 국어동화 6편을 자체 제작한다.
- [ ] 수학동화 6편을 자체 제작한다.
- [ ] 영어동화 6편을 자체 제작한다.
- [ ] 각 동화마다 이해 질문과 독후활동을 붙인다.

### 프로그램 구현

- [ ] 유치원 학습 놀이터 홈 화면을 만든다.
- [ ] 6세반과 7세반 선택 상태를 저장한다.
- [ ] 국어교실 화면을 만든다.
- [ ] 수학교실 화면을 만든다.
- [ ] 영어교실 화면을 만든다.

## 28. Cloudflare D1 병행 동기화 구현 결과

운영 사이트는 현재 프로젝트 DB인 브라우저 `localStorage`와 Cloudflare 전용 D1 DB를 병행 사용한다.

| 구분 | 구현 내용 |
|---|---|
| Cloudflare 전용 DB | `purunet_kindergarten_learning_db` |
| D1 바인딩 | `KINDERGARTEN_DB` |
| 동기화 API | `/api/kindergarten/sync` |
| 현재 프로젝트 DB | 브라우저 `localStorage`의 `kindergarten-playground-state-v3` |
| 저장 방식 | localStorage 저장 후 Cloudflare D1에 같은 스냅샷 병행 저장 |
| 읽기 방식 | 앱 시작과 새로고침 때 D1 스냅샷을 읽어 localStorage 상태와 병합 |
| 교사 데이터 | `teachers`, `classes` 테이블과 전체 snapshot payload에 저장 |
| 학습자 데이터 | `students`, `student_progress` 테이블과 전체 snapshot payload에 저장 |
| 학습용 데이터 | `learning_content`, `content_admin` 테이블에 코스 메타데이터와 운영 메모 저장 |
| API 보안 기준 | 같은 Cloudflare Pages origin의 PUT 요청만 저장 허용 |

동기화 파일은 다음과 같다.

| 파일 | 역할 |
|---|---|
| `js/cloud-sync.js` | 브라우저에서 D1 API로 스냅샷을 저장하고 읽는 클라이언트 |
| `_worker.js` | Cloudflare Pages에서 D1 API를 처리하는 Worker |
| `migrations/0001_kindergarten_cloud_sync.sql` | D1 테이블과 인덱스 생성 SQL |
| `wrangler.toml` | `purunet-kindergarten` Pages와 D1 바인딩 설정 |

검증은 `npm run check`, `npm run smoke`, 운영 URL Edge 점검, 원격 D1 SQL 조회로 진행했다. 운영 점검에서 교사 1명, 클래스 1개, 학생 1명, 학생 진행 1건, 학습 콘텐츠 6개가 D1에 저장되는 것을 확인한 뒤, 검증용 교사·학생·진행 데이터는 삭제하고 학습 콘텐츠 6개만 유지했다.
