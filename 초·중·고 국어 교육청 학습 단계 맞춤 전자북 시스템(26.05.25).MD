# 초·중·고 국어 교육청 학습 단계 맞춤 전자북 시스템

기준은 **2022 개정 국어과 교육과정**입니다. 국어과는 초·중·고 모두 **듣기·말하기, 읽기, 쓰기, 문법, 문학, 매체** 영역 중심으로 구성됩니다. ([창원대학교][1])

## 1. 전체 시스템 구조

```text
국어 전자북 시스템
→ 교육과정 성취기준 DB
→ 학년군별 콘텐츠
→ 인터랙티브 전자북
→ AI 문해력 분석
→ CBT/단원평가
→ 오답노트
→ 학원 관리자 LMS
```

## 2. 학습 단계별 콘텐츠 설계

| 구분   | 학습 목표          | 핵심 콘텐츠          | 전자북 기능            |
| ---- | -------------- | --------------- | ----------------- |
| 초1~2 | 한글 해득, 기초 읽기   | 낱말, 문장, 짧은 글    | 음절 클릭, 따라읽기, 받아쓰기 |
| 초3~4 | 문단 이해, 기초 문법   | 설명문, 이야기, 맞춤법   | 문장 구조 분석, 어휘 카드   |
| 초5~6 | 독해력, 요약, 토론    | 주장글, 문학, 매체     | 중심문장 찾기, 요약 훈련    |
| 중1~3 | 비판적 읽기, 문법, 문학 | 논설문, 시·소설, 국어문법 | 근거 찾기, 문법 하이라이트   |
| 고1   | 공통국어, 수능 기초    | 독서, 문학, 화법·작문   | 수능형 문제, 지문 분석     |
| 고2~3 | 수능·내신 심화       | 비문학, 문학, 언매, 화작 | 실전 모의고사, AI 해설    |

## 3. 초등 국어 전자북

### 초1~2

```text
한글 → 낱말 → 문장 → 짧은 글 → 말하기
```

주요 콘텐츠:

* 자음·모음 학습
* 받침 읽기
* 그림 단어
* 소리 내어 읽기
* 짧은 문장 만들기
* 받아쓰기

전자북 기능:

* 글자 클릭 시 발음 재생
* 음절 색상 분리
* 따라 읽기 녹음
* 그림 보고 낱말 맞히기
* AI 발음 피드백

---

### 초3~4

```text
문장 이해 → 문단 읽기 → 어휘 확장 → 기초 문법
```

주요 콘텐츠:

* 중심 낱말 찾기
* 문단 요약
* 높임 표현
* 맞춤법
* 동화·생활문 읽기

전자북 기능:

* 중심문장 하이라이트
* 어려운 어휘 팝업
* 맞춤법 퀴즈
* 문장 순서 배열
* AI 요약 연습

---

### 초5~6

```text
독해 → 요약 → 주장 파악 → 토론·글쓰기
```

주요 콘텐츠:

* 주장과 근거
* 설명문·논설문
* 문학 감상
* 매체 자료 읽기
* 독서 토론

전자북 기능:

* 주장/근거 색상 표시
* 문단별 핵심어 추출
* AI 요약 첨삭
* 토론 카드 생성
* 서술형 자동 피드백

---

## 4. 중등 국어 전자북

```text
독해력
+ 문법
+ 문학 감상
+ 서술형
+ 내신 대비
```

| 영역     | 콘텐츠                 | 기능              |
| ------ | ------------------- | --------------- |
| 읽기     | 설명문, 논설문, 기사        | 근거 찾기, 중심 내용 분석 |
| 문학     | 시, 소설, 수필, 희곡       | 인물·갈등·화자 분석     |
| 문법     | 품사, 문장성분, 음운, 단어 형성 | 문장 자동 분석        |
| 쓰기     | 주장글, 설명문, 감상문       | AI 첨삭           |
| 듣기·말하기 | 토의, 토론, 발표          | 발표 대본 생성        |
| 매체     | 광고, 뉴스, 영상 자료       | 매체 비판 읽기        |

중등 핵심 기능:

```text
교과서 지문 입력
→ 개념 설명
→ 빈칸 문제
→ 서술형 문제
→ 내신형 객관식
→ AI 오답 분석
```

## 5. 고등 국어 전자북

```text
공통국어
→ 독서
→ 문학
→ 화법과 작문
→ 언어와 매체
→ 수능 실전
```

| 과정      | 콘텐츠                   | 기능          |
| ------- | --------------------- | ----------- |
| 고1 공통국어 | 기본 독해, 문학, 문법         | 내신형 단원평가    |
| 독서      | 인문, 사회, 과학, 기술, 예술 지문 | 문단 구조 분석    |
| 문학      | 현대시, 고전시가, 현대소설, 고전소설 | 표현법·주제 분석   |
| 화법과 작문  | 발표, 토론, 보고서, 주장글      | 글쓰기 첨삭      |
| 언어와 매체  | 음운, 단어, 문장, 담화, 매체    | 문법 문제 자동 생성 |

고등 핵심 기능:

* 수능형 지문 분석
* 선지 근거 표시
* 오답 선지 해설
* 문학 표현법 자동 태깅
* 문법 개념별 반복 문제
* 실전 시간 제한 모의고사

## 최종 추천 방향

가장 좋은 구조는:

```text
교육청 국어 성취기준
+ 교과서 단원형 콘텐츠
+ AI 문해력 분석
+ CBT 평가
+ 서술형 첨삭
+ 학원 LMS
```

입니다.

즉, 단순 국어 전자책이 아니라 **초등 문해력 → 중등 내신 → 고등 수능국어**까지 이어지는 **국어 AI 자기주도학습 플랫폼**으로 제작하는 것이 가장 적합합니다.
