## 이번 보강 내용

1. 그림그래프 전용 벡터 렌더링 추가
   - `pictograph` 시각 자료 타입을 새로 만들었다.
   - 큰 그림 1개=10개, 작은 그림 1개=1개 구조를 실제 그림그래프 형태로 보여준다.
   - 12월 그림그래프 읽기, 비교, 나타내기 문제에 적용했다.

2. 원과 각도 시각 자료 보강
   - `circle-pattern` 타입을 추가해 원을 이용한 무늬를 벡터 그림으로 표시한다.
   - `angle-diagram` 타입을 추가해 각도 어림, 각도의 합, 회전 각도를 그림으로 표시한다.
   - 기존 2월 도형 돌리기 문제의 시각 자료도 실제 회전 각도 그림으로 바꾸었다.

3. PDF 세부 흐름에 맞춘 3학년 월별 재배치
   - 6월에서 PDF 흐름과 맞지 않던 길이·시간 문제를 줄이고 단위분수 중심으로 바꾸었다.
   - 7월과 9월에는 한 자리 수×몇십몇, 두 자리 수×두 자리 수 유형을 추가했다.
   - 8월에는 진분수·가분수·대분수 유형을 추가했다.
   - 11월에는 들이와 무게의 덧셈·뺄셈 표 문제를 추가했다.
   - 2월에는 평면도형 이동과 함께 막대그래프 눈금 읽기를 넣었다.

## 검증 기준

- 수학 부호: `+`, `-`, `×`, `÷`, `□`, `>`, `<`, `=`, `°` 표현을 사용
- 표: 자리값 표, 단위분수 비교표, 부분곱 표, 측정 덧셈·뺄셈 표 반영
- 그래프: 그림그래프, 막대그래프, 자료 비교 문제 반영
- 도형: 분수 막대, 원 도형, 원 무늬, 각도 그림, 대칭 도형 반영

## 최종 검증 결과

- 앱 생성기는 원문 문제를 그대로 복제하지 않고 같은 유형의 새 숫자 문제를 생성한다.
- PDF 원문 이미지 자체를 추출해 붙인 방식이 아니라, 학습 유형에 맞춘 벡터 기반 시각 자료로 재구성했다.
- `npm run lint`: 통과
- `npm run verify`: 전체 생성기 표본 검증 통과
- `npm run onefile`: 단일 HTML 재생성 완료
- 커리큘럼 연결 검사: 전체 316개 세부 메뉴의 `generate()` 호출 정상
- 생성 결과를 `푸르넷수학-연습.html`과 `모바일 홈페이지형 전자북(26.05.17)/study.html`에 반영했다.
