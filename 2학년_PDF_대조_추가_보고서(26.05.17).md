## 재검증에서 발견한 보완점과 처리

| 항목 | 기존 상태 | 처리 내용 |
|---|---|---|
| 길이 단원 | 길이가 표 또는 문장으로만 보이는 문제가 있었다. | `ruler` 시각 타입을 추가해 cm/mm 자 눈금과 측정 구간을 SVG로 표시했다. |
| 시각 단원 | 시각 읽기가 표 중심이라 PDF의 시계형 문제 느낌이 약했다. | `clock` 시각 타입을 추가해 시침·분침·눈금이 있는 아날로그 시계를 표시했다. |
| 선과 직각 | 선분·반직선·직선, 직각 도형이 설명 문장 중심이었다. | `line-type`, `right-angle` 타입을 추가해 선의 끝점·화살표·직각 표시를 벡터로 표현했다. |
| 곱셈 배열 | 배열 곱셈이 표로만 제시되어 실제 묶음 감각이 부족했다. | `object-array` 타입을 추가해 줄과 칸을 점 배열로 보여 주도록 바꿨다. |
| 무늬 규칙 | 무늬 규칙이 텍스트 기호로만 보였다. | `shape-pattern` 타입을 추가해 삼각형·원·사각형·색 무늬를 벡터 도형으로 보여 주도록 바꿨다. |
