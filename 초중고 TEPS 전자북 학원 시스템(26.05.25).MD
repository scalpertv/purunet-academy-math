아래처럼 **초등–중등–고등 TEPS 전자북 학원 시스템**으로 확장하면 됩니다.

## 1. 전체 구조

| 구분    | 초등 TEPS       | 중등 TEPS             | 고등 TEPS               |
| ----- | ------------- | ------------------- | --------------------- |
| 목표    | 영어 기초 + 시험 감각 | 유형 적응 + 내신/TEPS 병행  | 실전 TEPS + 수능·대학 영어 연결 |
| 대상    | 초3~초6         | 중1~중3               | 고1~고3                 |
| 난이도   | 그림·음성·패턴      | 문법·독해·청해 유형         | 고난도 어휘·추론·실전 모의고사     |
| 핵심 방식 | 게임형 전자북       | 문제풀이형 전자북           | 실전 시험형 전자북            |
| 결과    | 영어 기본기        | TEPS Junior/기초 TEPS | TEPS 점수 향상 + 수능 독해력   |

---

## 2. 초등 TEPS 과정

| 영역 | 구성                       |
| -- | ------------------------ |
| 단어 | 그림 단어, 발음, 예문, 카드 퀴즈     |
| 문법 | be동사, 일반동사, 의문문, 시제, 조동사 |
| 듣기 | 단어 듣기, 문장 듣기, 그림 고르기     |
| 독해 | 짧은 문장, 3~5문장 지문          |
| 활동 | 단어 게임, 문장 배열, 받아쓰기       |

핵심은 **재미 + 반복 + 기초 패턴**입니다.

---

## 3. 중등 TEPS 과정

| 영역 | 구성                          |
| -- | --------------------------- |
| 어휘 | 중등 필수어휘 + TEPS 빈출어휘         |
| 문법 | 시제, 수동태, 부정사, 동명사, 관계사, 접속사 |
| 청해 | 짧은 대화, 상황 파악, 핵심 정보 찾기      |
| 독해 | 주제, 제목, 빈칸, 세부 내용, 추론       |
| 실전 | 단원 테스트, 미니 모의고사, 오답노트       |

핵심은 **유형 적응 + 내신 영어와 병행**입니다.

---

## 4. 고등 TEPS 과정

| 영역    | 구성                              |
| ----- | ------------------------------- |
| 고급 어휘 | TEPS 고빈출 어휘, 동의어, 반의어, 숙어, 학술어휘 |
| 고급 문법 | 분사구문, 가정법, 관계사 심화, 병렬구조, 도치, 생략 |
| 청해    | 긴 대화, 강의형 듣기, 의도 파악, 추론         |
| 독해    | 빈칸 추론, 논리 전개, 어조, 목적, 장문 독해     |
| 실전    | 실전 모의고사, 시간 제한 풀이, 점수 예측        |
| 수능 연계 | 수능 독해 유형과 TEPS 독해 유형 비교 학습      |

고등 과정은 **점수형 시스템**으로 설계해야 합니다.

예시:

| 레벨                | 목표               |
| ----------------- | ---------------- |
| High Basic        | TEPS 기초 유형 적응    |
| High Intermediate | 실전 문제풀이 속도 향상    |
| High Advanced     | 고득점 전략·시간 관리     |
| 실전반               | 모의고사 + 약점 집중 클리닉 |

## 10. 최종 개발 방향

가장 좋은 구조는 다음입니다.

**초등: 게임형 TEPS 전자북**
**중등: 유형별 문제풀이 TEPS 전자북**
**고등: 실전 모의고사 + AI 약점 분석 TEPS 시스템**

즉, 단순 전자책이 아니라
**“TEPS 전자북 + 문제은행 + 오답노트 + 학원관리 + AI 해설”** 플랫폼으로 만들어야 상품성이 높습니다.
